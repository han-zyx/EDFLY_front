Empowering Real-Time Data Streaming from Edge to Cloud

About

EDFLY is an advanced edge-to-cloud data streaming and monitoring solution, integrating Apache Flink for stream processing, YOLOv8 for object detection, and Prometheus-Grafana for monitoring. It enables seamless, low-latency data flow, making it ideal for real-time analytics in smart surveillance, IoT, and industrial automation.

Why

As real-time data processing demands grow, traditional cloud architectures face latency, bandwidth, and reliability challenges. EDFLY overcomes these by leveraging edge computing, enabling faster decision-making, scalability, and secure, efficient data streaming from multiple edge devices.

Research Gap

Most real-time data processing solutions rely on cloud-based architectures, causing latency and network dependency issues. While edge computing mitigates these, integrated frameworks combining stream processing, real-time object detection, and monitoring are scarce. Many existing approaches also lack seamless edge-to-cloud integration, leading to inefficiencies in data management and decision-making.
