About

What is EDFLY

EDFLY is a real-time data streaming and monitoring solution designed for edge computing. It combines Apache Flink for stream processing, YOLOv8 for object detection, and Prometheus-Grafana for monitoring. EDFLY enables fast, efficient, and secure data processing, making it ideal for IoT, smart surveillance, and industrial automation.

Key Features

  • Seamless edge-to-cloud data streaming
  • Low-latency processing with Apache Flink
  • Real-time object detection with YOLOv8
  • Comprehensive monitoring using Prometheus and Grafana

Technology Stack

EDFLY integrates Apache Flink, YOLOv8, Prometheus, Grafana, and cloud-based analytics to provide real-time streaming, detection, and monitoring in edge computing environments. The combination of these technologies enables efficient data management and decision-making.
